# 后端管理员平台图片上传功能说明

## 功能概述

后端管理员平台已集成图片上传功能，支持在商品管理模块中上传商品图片。图片将上传到服务器本地存储，并通过HTTP接口访问。

## 已开发的功能

### 1. 图片上传组件

**文件位置：** `src/components/ImageUpload.vue`

**功能特点：**
- 支持单个图片上传
- 支持多种图片格式（jpg、jpeg、png、gif、webp）
- 文件大小限制（默认10MB）
- 上传进度显示
- 自动处理图片URL拼接
- 错误提示和处理

**使用方式：**

```vue
<template>
  <ImageUpload
    v-model="form.image"
    placeholder="点击上传图片"
    tip="建议尺寸：800x800px，支持 jpg、png、gif、webp 格式"
  />
</template>

<script setup lang="ts">
import ImageUpload from '@/components/ImageUpload.vue'
</script>
```

### 2. 商品管理集成

**文件位置：** `src/views/product/list/components/ProductFormDialog.vue`

**功能说明：**
- 在新增商品和编辑商品表单中集成图片上传
- 支持上传商品主图
- 上传成功后自动保存图片路径到表单数据
- 提交商品信息时一起提交图片URL

## 技术架构

### 图片上传流程

```
┌─────────────┐
│  选择图片   │
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  前端验证   │  格式、大小
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  调用上传接口 │  POST /api/upload/image
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  后端保存   │  D:/项目图标/年月/文件名
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  返回URL   │  /images/年月/文件名
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  保存到表单 │  form.image = URL
└─────────────┘
```

### API接口

**上传接口：** `POST /api/upload/image`

**请求参数：**
- file: MultipartFile（图片文件）

**响应格式：**
```json
{
    "code": 200,
    "msg": "success",
    "data": {
        "url": "/images/2026-03/product_1743401234567_abc123de.jpg",
        "filename": "product_1743401234567_abc123de.jpg",
        "size": 102400,
        "uploadTime": "2026-03-31T14:30:00",
        "originalFilename": "product.jpg",
        "fileType": "jpg"
    }
}
```

### 图片访问

**访问URL：** `http://localhost:8081/images/{year-month}/{filename}`

**示例：**
```
http://localhost:8081/images/2026-03/product_1743401234567_abc123de.jpg
```

## 使用说明

### 1. 后端启动

确保后端服务已启动：
```bash
cd backend-java
mvn spring-boot:run
# 或运行 jar 包
java -jar target/luxury-watch-backend.jar
```

后端服务地址：`http://localhost:8081`

### 2. 前端启动

启动后端管理平台：
```bash
cd backend-admin
npm run dev
```

访问地址：`http://localhost:5173`（默认端口）

### 3. 图片上传操作

#### 新增商品

1. 进入商品管理页面
2. 点击"新增商品"按钮
3. 填写商品基本信息（名称、编码、分类、价格、库存等）
4. 在"商品图片"区域点击上传按钮
5. 选择要上传的图片文件（支持 jpg、png、gif、webp 格式，大小不超过10MB）
6. 图片上传成功后会自动显示在上传区域
7. 继续填写商品描述等信息
8. 点击"确定"按钮提交

#### 编辑商品

1. 进入商品管理页面
2. 点击已有商品的"编辑"按钮
3. 在表单中可以看到已上传的商品图片
4. 如需更换图片，点击上传区域重新上传
5. 修改其他信息
6. 点击"确定"按钮保存修改

### 4. 图片查看

上传成功的图片会在以下位置显示：
- 商品列表：商品图片列
- 商品表单：上传区域
- 商品详情页面（需单独开发）

## 配置说明

### 后端配置

**配置文件：** `backend-java/src/main/resources/application.yml`

```yaml
upload:
  # 图片存储根目录
  image-path: D:/项目图标
  # 支持的图片格式
  allowed-extensions: jpg,jpeg,png,gif,webp
  # 最大文件大小（单位：MB）
  max-file-size: 10
  # 图片访问基础URL
  base-url: /images
```

### 前端配置

**环境配置：** `backend-admin/.env.development`

```env
VITE_APP_API_BASEURL = http://localhost:8081
```

**生产环境配置：** `backend-admin/.env.production`

```env
VITE_APP_API_BASEURL = http://your-server:8081
```

## 约束条件

### 文件格式限制

| 格式 | 扩展名 | 支持状态 |
|-----|-------|---------|
| JPEG | .jpg, .jpeg | ✅ 支持 |
| PNG | .png | ✅ 支持 |
| GIF | .gif | ✅ 支持 |
| WebP | .webp | ✅ 支持 |

### 文件大小限制

| 配置项 | 值 |
|-------|-----|
| 单个文件最大大小 | 10 MB |
| 建议图片尺寸 | 800x800 px |

### 存储规则

- 图片按上传年月自动分类存储
- 文件名自动生成（时间戳+UUID+扩展名）
- 存储路径：`D:/项目图标/年月/文件名`

## 常见问题

### 1. 图片上传失败

**可能原因：**
- 后端服务未启动
- 网络连接问题
- 文件格式不符合要求
- 文件大小超过限制
- 服务器存储目录不存在或无写入权限

**解决方案：**
1. 确保后端服务已启动并运行在8081端口
2. 检查网络连接
3. 确认图片格式和大小符合要求
4. 检查 `D:/项目图标` 目录是否存在
5. 检查目录写入权限

### 2. 图片无法显示

**可能原因：**
- 图片URL格式不正确
- 后端静态资源映射未正确配置
- 图片文件不存在或被删除

**解决方案：**
1. 检查浏览器开发者工具中的图片URL
2. 确保使用完整URL访问图片
3. 确认后端服务正常运行
4. 检查图片是否存在于服务器目录

### 3. 上传进度不显示

**说明：** 当前组件暂不支持上传进度显示，这是后续可优化项。

## 代码复用

### 在其他模块中使用图片上传

如果需要在其他表单中复用图片上传功能，只需：

1. **导入组件：**
```typescript
import ImageUpload from '@/components/ImageUpload.vue'
```

2. **使用组件：**
```vue
<el-form-item label="图片">
  <ImageUpload
    v-model="formData.imageUrl"
    placeholder="上传图片"
    tip="支持 jpg、png、gif、webp 格式"
  />
</el-form-item>
```

3. **接收上传成功事件（可选）：**
```vue
<ImageUpload
  v-model="formData.imageUrl"
  @upload-success="handleUploadSuccess"
  @upload-error="handleUploadError"
/>
```

### 禁用上传功能

如果需要禁用上传功能（例如查看模式）：

```vue
<ImageUpload
  v-model="formData.imageUrl"
  :disabled="true"
/>
```
