# 图片上传调试说明

## 如何使用调试信息

### 1. 打开浏览器开发者工具

1. 按 `F12` 或 `Ctrl+Shift+I`（Windows）打开开发者工具
2. 点击 **Console（控制台）** 标签

### 2. 执行图片上传操作

1. 进入商品管理页面
2. 点击"新增商品"或"编辑商品"
3. 上传一张图片

### 3. 查看控制台输出

上传过程中，控制台会输出以下调试信息：

#### 📤 上传开始
```
📤 [ImageUpload] 开始上传文件: heroImage.jpg 大小: 511881 类型: image/jpeg
✅ [ImageUpload] 文件验证通过，准备上传
```

#### 🔍 上传成功回调
```
🔍 [ImageUpload] 上传成功回调触发
📦 [ImageUpload] 原始 response: {...}  ← 查看完整 axios response 对象
📋 [ImageUpload] response 结构: {...}  ← 查看解析后的结构
🎯 [ImageUpload] 解析后的 resData: {...}  ← 查看 API 返回数据
✅ [ImageUpload] 上传成功，图片URL: /images/2026-03/xxx.jpg
```

#### 🔧 组件初始化
```
🔧 [ImageUpload] 组件初始化: {...}  ← 查看初始配置
```

#### 🖼️ 计算图片URL
```
🖼️ [ImageUpload] 计算 fullImageUrl: {...}  ← 查看 URL 计算过程
```

## 常见问题诊断

### ❌ 问题1：上传成功后一直 loading

**查看这个信息：**
```
🔍 [ImageUpload] 上传成功回调触发
📋 [ImageUpload] response 结构:
  - response.data: undefined ❌
  - response.status: 200
```

**如果 `response.data` 是 `undefined`：**

说明 API 拦截器没有正确返回数据，或者请求根本没有经过拦截器。

**可能原因：**
1. 文件上传请求没有走 axios 拦截器
2. 请求被浏览器缓存
3. 网络错误

### ❌ 问题2：图片 URL 为空

**查看这个信息：**
```
🎯 [ImageUpload] 解析后的 resData: {code: 200, data: {url: undefined}} ❌
```

**可能原因：**
1. 后端返回的数据结构不对
2. URL 字段名不是 `url`

### ❌ 问题3：fullImageUrl 计算错误

**查看这个信息：**
```
🖼️ [ImageUpload] 计算 fullImageUrl:
  - imageUrl.value: /images/2026-03/xxx.jpg  ✅
  - baseUrl: http://localhost:8081  ✅
  - 计算结果: http://localhost:8081/images/2026-03/xxx.jpg  ✅
```

如果计算结果不对，检查 `baseUrl` 配置。

### ❌ 问题4：via.placeholder.com 错误

**查看这个信息：**
```
🔧 [ImageUpload] 组件初始化:
  - props.modelValue(原始值): https://via.placeholder.com/300x300 ❌
```

**问题原因：**
数据库中保存了旧的占位图 URL。

**解决方案：**
1. 执行清理 SQL：
```sql
UPDATE `product` SET `image` = NULL WHERE `image` LIKE '%placeholder%';
```
2. 重新上传图片

## 如何根据调试信息修复问题

### 1. 如果看到 `response.data` 是 `undefined`

**原因：** 请求可能没有经过 axios 拦截器

**检查：**
1. 查看 Network 面板，确认请求 URL
2. 检查是否通过代理 `/proxy/`
3. 检查 API 拦截器配置

### 2. 如果看到 `resData.code` 不是 `200`

**原因：** API 返回了错误

**检查：**
1. 查看 `resData.msg` 获取错误信息
2. 检查后端日志

### 3. 如果看到 `via.placeholder.com`

**原因：** 数据库中有旧的占位图 URL

**解决：**
```sql
UPDATE `product` SET `image` = NULL WHERE `image` LIKE '%placeholder%';
```
